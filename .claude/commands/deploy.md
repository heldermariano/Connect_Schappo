Prepare para deploy com Docker + Traefik:

1. Verifique/crie Dockerfile (multi-stage, node:20-alpine)
2. Verifique/crie docker-compose.yml com labels Traefik para connect.clinicaschappo.com
3. Crie .dockerignore
4. Rode `npm run build` — verificar erros
5. Rode `docker compose build` — verificar imagem
6. Crie `docs/GUIA_DEPLOY_PRODUCAO.md`
